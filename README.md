# nokia0024 

### Projet de fin d'études réalisé par Matthieu Barbe, Bastien Matis et moi-même.

 Application mobile qui réalise l'un de nos souhaits les plus fous : voyager dans le temps 🚀. On vous fait plonger dans un temps où les smartphones n'existaient pas... mais où les Nokia 3310 étaient partout ! Faites remonter des souvenirs (en jouant à Snake, par exemple), mais testez aussi des fonctionnalités dont le vous du passé n'aurait pu que rêver : la lecture de fichiers MP3, un système de messagerie avec les autres utilisateurs de l'application. 
 
 ### Tech 
 Front-end : React Native, Typescript
 Back-end : Node, Prisma ORM 
 BDD : Mariadb
 
Déploiement du back-end à l'origine sur Scaleway. Build d'un APK et déploiement géré à la base avec une CI/CD. On avait commencé à faire la demande pour déployer sur Play Store 🚀
